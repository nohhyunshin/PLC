## PLC를 사용한 엘리베이터 기능 구현

---

### 프로젝트 개요
본 프로젝트는 PLC 기반 제어 로직과 C# 상위 제어 프로그램을 연동하여 엘리베이터의 기본 동작을 구현한 자동화 시스템 실습 프로젝트로, 하드웨어와 소프트웨어의 역할을 분리하고 신호 흐름과 상태 전이를 명확하게 설계하는 것을 목표로 하였다.

---

### 시스템 구성
- PLC
  - 층 호출 입출력 처리

- C# 프로그램
  * 사용자 인터페이스 구현 (엘리베이터 층수 버튼 및 층간 이동)
  - 현재 층과 이동 상태 표현

---

### 주요 기능 설명
· 층 호출 처리 로직
· 이동 방향 판단 (호출된 층과 현재 층 비교)
· 실시간 상태 표시 (이동 중 상하 방향 표시)
· 엘리베이터 개폐 시 효과음 (개폐 시 다른 효과음으로 직접적인 효과)
